# x-auto-plugin

Claude Code plugin for semi-automated X (Twitter) operation. Generates personalized tweet drafts based on your brand voice, post history, and market trends.

## Features

- **brand-voice-based draft generation**: Personalized tweets from your own voice
- **Progressive enhancement**: Works without X API; richer with API connected
- **Onboarding skill**: Generates initial brand voice from sample tweets

## Usage

1. Run `x-setup` to generate your `brand-voice.md`
2. Say "今日のツイート作って" to generate drafts
3. Pick/edit drafts, then `x-post` to publish
