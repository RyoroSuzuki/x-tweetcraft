# x-tweetcraft

**あなたの分身が、あなたの代わりにXツイートを考えてくれるツール。**

AIが書いたと思わせない、「本物のあなたがそこで考えた」かのようなツイート下書きを生成します。

---

## これは何？

「X（Twitter）の投稿、自分で毎回考えるのは大変。でもAIに任せると『AI感』が出てしまって、自分らしくない」

そんな人のために、**あなたの口調・価値観・考え方を深く学習して、あなたの分身となってツイート案を作る** ツールです。

ツールは以下の情報をあなたから学びます：
- あなたの過去のツイート
- あなたのキャリア・原体験・価値観（対話インタビューで掘り下げ）
- あなたの届けたい人・発信したいテーマ

この「分身」は時間とともに賢くなり、使うほど **「本物のあなたが考えたとしか思えない投稿」** を作れるようになっていきます。

### 何ができるの？

- 🎯 **あなたの分身があなたらしい下書きを5〜10個生成**
- 🧠 **インタビューで価値観・原体験を掘り下げ** て、分身に深みを注入
- 📊 **自分の過去投稿を分析** して「何が伸びたか」を学習（X API接続後）
- 🔍 **今X上で伸びてる型をリサーチ** してあなたの文脈に翻訳（X API接続後）
- 📅 **週次戦略レビュー** で来週の方向性を提案（X API接続後）
- 🔄 **使うほど賢くなる** — 採用/却下された下書きから分身が自動学習

### 何をしないの？

- ❌ 勝手に投稿はしません（下書きを見せるだけ。投稿はあなたが確認してから）
- ❌ あなたの個人情報をどこにも送りません（すべてあなたのPC内で動く）
- ❌ 「AIが書いた感」を出しません（あなたの文体・語尾・視点をそのまま再現します）

## インストール

**必要なもの:** [Claude Code](https://claude.com/product/claude-code)（未導入ならまずこちらをインストール）

### 方法A: Claudeに任せる（最も簡単）

Claude Codeで以下のように話しかけてください：

```
x-tweetcraftをインストールしたい
```

→ Claudeが以下を代行します：
1. **保存先をあなたに確認**（現在のフォルダ、ドキュメント、ホーム等）
2. GitHubからツール一式をダウンロード
3. 関連ツール（おすすめのClaude Code公式スキル）のチェックと導入案内
4. そのまま使える起動方法を案内

### 方法B: 自分でダウンロード（ターミナル操作）

ターミナルで好きな場所に移動して、ダウンロード → その中に移動して Claude Code 起動：

```bash
cd <お好きな場所>        # 例: cd ~/Documents や cd ~/my-projects
git clone https://github.com/RyoroSuzuki/x-tweetcraft.git
cd x-tweetcraft
claude --plugin-dir .
```

**これだけ。** x-tweetcraft フォルダの中で Claude Code を起動すれば、プラグインもユーザーデータも全部そこで完結します。

> 💡 **あなたのデータ（brand-voice.md や personal-info/）はx-tweetcraftフォルダの中に作られます。**
> git で管理されるのはプラグイン本体だけ（ユーザーデータは自動で除外済み）なので、安心して一箇所にまとめて使えます。

## はじめての使い方

Claude Codeを起動したら、こう話しかけてください：

```
x-tweetcraftをセットアップして
```

以下の流れで進みます：

1. **Xアカウント情報を伝える** — URLか `@ハンドル`
2. **ツイートを5〜10件教える** — 「これは自分らしい」と思うものを。[Claudeブラウザ拡張](https://claude.com/chrome)があれば自動抽出プロンプトも提示されます
3. **あれば追加資料を渡す** — X運用方針メモ・ブランディング資料など（なくてもOK）
4. **あなた用のブランドボイス雛形が自動生成される** — トーン・テーマ・型・NG事項
5. **深掘りインタビューへ進むか選ぶ** — 5分のRound 1だけでもOK

関連する Claude Code 公式スキルがまだ入っていない場合は、ここで自動的に案内されます。

## 日常の使い方

### ツイートを作って投稿する

```
今日のツイート作って
```

→ あなたらしい下書きが5〜10個、根拠付きで出てきます。

気に入ったものを選んで返答：

```
3番でいく。ただし文末の「〜だ」を「〜だな」に変えて
7番は次回用にキープ
2番は違う、上から目線すぎ
```

→ X API接続済なら **その場で投稿**、未接続ならクリップボードにコピーされます。

選んだ理由や修正内容は自動で学習され、次回の生成から反映されます。使うほど分身があなたらしくなっていきます。

### 投稿の精度を上げたい時

「下書きが浅い」「自分らしくない」と感じたら、以下の手段があります：

```
もっとあなたのことを知って精度を上げたい
```

→ Claudeが状況を聞いて、以下のどれかを提案します：

- **対話インタビュー** — あなたの職歴・価値観・原体験を聞いて、分身を深める（5〜30分、分割OK）
- **X API連携** — 過去投稿の実績データや市場トレンドを分析に反映（下の「応用」参照）
- **参考ツイートを追加** — 「こういう投稿が好き」を伝えて反映してもらう

## よくある質問

### Q: 自分の投稿データはどこに送られますか？
A: どこにも送られません。すべてあなたのPC内で、あなたのClaude Code契約のAIとだけやり取りします。作者（Ryoro）や第三者のサーバーは経由しません。

### Q: X以外のSNS（Instagram, Threads, Note等）でも使えますか？
A: 現時点ではX特化ですが、ブランドボイスの仕組みは汎用なので将来対応の可能性あり。

### Q: MCPサーバーのコードが動かなくなったら？
A: Claude Codeに相談しながら自分でカスタマイズしてください。コードは約150行で、読みやすく書かれています。「このエラー出たんだけど」と伝えれば一緒に直せます。

### Q: 困った時の全般的な相談先は？
A: Claude Code自体があなたの相談相手です。「x-tweetcraftのこの機能が動かない」「こういう挙動にしたい」と話しかければ、コードを読んで対応してくれます。

## プラグインに含まれるもの

**9個のスキル:**
- `x-setup` — 初回オンボーディング
- `x-interview` — 深掘りインタビュー（分割・再開可）
- `x-draft` — 下書き5〜10案生成
- `x-improve` — 下書きが浅いときに原因診断＆ルーティング
- `x-reflect` — 採用/却下から自動学習
- `x-schedule` — 自動化設定
- `x-connect-api` — X API接続ガイド（ClaudeがPython環境セットアップも代行）
- `x-analyze-posts` — 過去投稿の深い分析（API必要）
- `x-research-trends` — 市場トレンド研究（API推奨）

**1個のエージェント:**
- `x-content-strategist` — 週次戦略レビュー

**1個のMCPサーバー（自作）:**
- X API v2 を呼ぶPythonコード約150行。あなた自身が読んで監査できるサイズ。

> ⚠️ MCPサーバーは作者の実X APIキーでの動作テスト未完了です。動かない箇所があればClaude Codeと一緒にカスタマイズしてください。
