# X Developer API プラン・コスト詳細（2026年4月時点）

> **用語確認:** ここで言う「プラン」は **X Developer API（開発者向けAPI）のプラン** です。
> X Premium（青バッジ付きの一般ユーザー向けサブスク）とは別物です。
> x-tweetcraft は Developer API のみ使用するため、X Premium は不要です。

**重要:** X API の料金・制限は頻繁に変わります。申請時は必ず公式サイトで最新情報を確認してください。
- 公式プランページ: https://docs.x.com/x-api/getting-started/pricing
- Developer Console: https://console.x.com

## 🆕 2026年の大きな変化: Pay-Per-Use（従量課金）登場

2026年2月6日、X API に **Pay-Per-Use（従量課金）モデル** が正式ローンチされました。
これまで月額$200の Basic プラン一択だった小規模ユーザーが、**月数ドル〜で使える** ようになりました。

**x-tweetcraft を使う個人ユーザーにとってはこのモデルが最適** です。

## プラン一覧

### Pay-Per-Use（従量課金・2026年新モデル・個人推奨）

**料金体系:** 使った分だけ支払い（プリペイド式クレジット購入）

**エンドポイント別単価（2026年4月時点の公開情報ベース）:**

| 操作 | 単価 | 用途 |
|------|------|-----|
| Post creation（投稿） | **$0.010 / 件** | ツイート投稿 |
| Post read（ツイート取得・検索） | **$0.005 / 件** | 自分の投稿取得、トレンド検索、特定ツイート取得 |
| User lookup（ユーザー情報取得） | **$0.010 / 件** | ユーザー情報取得 |

※料金は Developer Console で最新を確認すること。

**便利機能:**
- ✅ **月額上限設定（Spending Cap）** — 使いすぎ防止の自動ストップ
- ✅ **24時間重複課金回避** — 同じリソースを再取得しても再課金されない
- ✅ **オートトップアップ設定** — 残高不足時に自動チャージ
- ✅ **xAIクレジット還元** — 月$200以上の利用で最大20%還元

**制限:**
- 月200万投稿読み取りが上限（超えると Enterprise 必須）

**x-tweetcraft の各機能コスト概算:**

| 機能 | 処理内容 | 1回あたり |
|------|---------|---------|
| x-draft（API連携時） | 自分30件+検索30件 = 60 reads | 約 $0.30 |
| x-post | 1 post | $0.010 |
| x-analyze-posts | 自分100件取得 | 約 $0.50 |
| x-research-trends | 検索100件 | 約 $0.50 |
| x-content-strategist agent | 分析＋検索 ≈150件 | 約 $0.75 |

### Free プラン（無料・テスト向け）

**月額**: $0

**できること:**
- 自分のアカウントからツイート投稿: **月1,500件まで**（1日約50件）
- 読み取り系: **15分に1リクエスト**（極めて制限強い）
- ユーザー情報取得: 限定的

**できないこと:**
- ツイート検索（Search APIなし）
- トレンド取得
- 本格的な分析

**x-tweetcraft との相性:**
- ⭕ `x-post` — 使える（月1,500件は十分）
- ❌ `x-analyze-posts` — 読み取り制限が厳しすぎて実質使えない
- ❌ `x-research-trends` — 検索APIなしで使えない
- ❌ `x-refresh-brand-voice` — 投稿取得件数が足りない

---

### Basic プラン（固定月額・月額運用のヘビーユーザー向け）

**月額**: **$200 USD**（約30,000円）/ 年契約なら**$175/月 = $2,100/年**

**できること:**
- ツイート投稿: 月10,000件
- ツイート取得・検索: 十分な枠
- メトリクス詳細取得: 可能

**x-tweetcraft との相性:**
- ⭕ 全機能フル活用可能
- ⭕ 毎日大量に投稿・分析するヘビーユーザー向け

**Pay-Per-Use vs Basic の損益分岐:**
- 月200件以下の投稿・軽い分析: **Pay-Per-Use圧勝**
- 月数千件の投稿・大量分析: **Basicが経済的**

---

### Pro プラン（法人向け）

**月額**: **$5,000 USD**（約75万円）

**特徴:**
- ツイート取得: 月100万件
- 全文検索API（フル履歴）
- 企業・研究機関向け

**x-tweetcraft ユーザーの推奨:** 個人・中小規模では不要。

---

### Enterprise プラン（大企業向け）

**月額**: **$42,000+ USD**（約630万円〜）

大企業・データ分析会社向け。個人利用では絶対に不要。

## よくある質問

### Q: Pay-Per-Use の実際の料金はどう確認する？
A: https://console.x.com/ にログインして「Pricing」ページで、エンドポイント別の最新料金が確認できます。料金は予告なく変更される可能性があります。

### Q: Free で我慢できる？
A: 「下書き生成＋手動コピペ投稿」ならAPIすら不要（0円）。「自動投稿だけ欲しい」ならFreeで月1,500件までOK。「分析機能」が欲しくなったらPay-Per-Useへ。

### Q: 月$200が高い？
A: 2026年からは**Pay-Per-Useで月数ドル〜数十ドル**で済みます。Basicが必要な人は限られます。

### Q: クレジットは使い切らないと損？
A: Pay-Per-Useは**使った分だけ**課金されるので、買ったクレジットが余っても問題なし。月額契約と違って「使わないと損」にならない。

### Q: 急にコストが爆発する心配は？
A: **月額上限設定**を必ず有効にする。超えたらAPIが止まるので、意図しない課金爆発を防げます。

### Q: xAIクレジット還元って何？
A: 月$200以上API使うと、その分の最大20%がxAI（Grok APIなど）のクレジットとして戻ってきます。x-tweetcraftユーザーの多くは$200/月に届かないので、この特典は考慮しなくてOK。
