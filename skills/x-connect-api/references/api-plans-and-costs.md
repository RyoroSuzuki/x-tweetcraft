# X Developer API プラン・コスト詳細

**重要:** X API の料金・制限は頻繁に変わります。申請時は必ず https://developer.x.com/en/portal/products で最新情報を確認してください。

本ドキュメントは 2024年後半時点の情報に基づいています。

## プラン一覧と料金

### Free プラン（無料）

**月額**: $0

**できること:**
- 自分のアカウントからツイート投稿: **月1,500件まで**（1日約50件）
- 自分のツイート取得: **月100件まで** ← **非常に少ない**
- ユーザー情報取得: 限定的

**できないこと:**
- ツイート検索（他人のツイート含む）
- トレンド検索
- メトリクス詳細取得（いいね数・RP数は取れるが、インプレッションは制限）

**x-tweetcraft との相性:**
- ⭕ `x-post` — 使える
- ❌ `x-analyze-posts` — 月100件だと実質使えない
- ❌ `x-research-trends` — 検索APIが無いため使えない
- ❌ `x-refresh-brand-voice` — 投稿取得件数が足りない

**Freeでx-tweetcraftを使うと:**
実質 L1（生成）+ 投稿自動化のみ。L2・L3機能は動かない。

---

### Basic プラン（推奨）

**月額**: **$200 USD**（約30,000円）

**できること:**
- ツイート投稿: **月3,000件まで**
- ツイート取得: **月50,000件まで**
- ツイート検索: **月10,000件まで** ← これが重要
- メトリクス詳細取得: 可能

**x-tweetcraft との相性:**
- ⭕ `x-post` — 余裕
- ⭕ `x-analyze-posts` — 月100投稿分析しても余裕
- ⭕ `x-research-trends` — 月10,000検索で十分
- ⭕ `x-refresh-brand-voice` — 過去投稿100件以上分析可能
- ⭕ `x-content-strategist` エージェント — 週次レビューも動く

**Basic推奨の理由:** x-tweetcraftが本領発揮するには Basic が必要。

---

### Pro プラン（個人には高すぎる）

**月額**: **$5,000 USD**（約75万円）

**特徴:**
- ツイート投稿: 月300,000件
- 全文検索API（フル履歴）
- 企業・研究機関向け

**x-tweetcraftユーザーの推奨:** 個人利用では不要。

---

### Enterprise プラン

**月額**: カスタム（数十万〜数百万円）

大企業・データ分析会社向け。個人利用では絶対に不要。

## コスト感：あなたの運用にいくらかかるか？

### ケースA: 月1万フォロワー未満の個人発信者
**想定運用:**
- 毎日1-3投稿
- 月次で自分の投稿振り返り
- 週次でトレンド分析

**推奨プラン:** Basic ($200/月)
**月額コスト目安:** 約30,000円

### ケースB: 個人ブランディング強化中の経営者
**想定運用:**
- 毎日3-5投稿
- 週次で投稿分析・戦略レビュー
- 月次でbrand-voiceリフレッシュ

**推奨プラン:** Basic ($200/月)
**月額コスト目安:** 約30,000円

### ケースC: まずはお試しで使いたい
**想定運用:**
- 下書き生成ツールとして手動運用
- 自動投稿・分析は不要

**推奨プラン:** Free ($0)
**月額コスト目安:** 0円

### ケースD: 複数アカウント運用・代行業務
**想定運用:**
- アカウント複数
- 月数千投稿レベル
- 詳細分析・顧客レポート

**推奨プラン:** Basic × 必要アカウント数、または上位プラン検討
**月額コスト目安:** 30,000円 × アカウント数

## コスト判断フロー

```
Q1: x-tweetcraftに期待する機能は？
├─ 下書き生成のみ（手動投稿OK）
│    → Free ($0) で十分。API申請も不要。
│
├─ 下書き生成 + 自動投稿
│    → Free ($0)。投稿数月50件まで。
│
├─ 下書き生成 + 投稿分析 + 市場調査 + 自動化
│    → Basic ($200/月 ≈ 30,000円) が必須。
│
└─ 複数アカウント・大規模運用
     → Basic複数 or 上位プラン検討。
```

## 申請時の注意

- 最初から Basic に申し込まなくてOK。**まず Free で始めて、必要を感じたら Basic アップグレード** がコスト的に安全。
- Basic申請には「承認」は不要（課金するだけで使える）。Freeは申請承認が必要。
- プラン変更はダッシュボードからいつでも可能。

## よくある質問

### Q: Freeで我慢できる？
A: 下書き生成ツール＋手動コピペ投稿だけなら、APIすら不要で x-tweetcraft は動きます。「自動化」「分析」「戦略」が欲しいなら Basic 一択。

### Q: $200/月は高い？
A: 時給3,000円の人が x-tweetcraft で月10時間節約できれば元は取れる計算。発信が事業成長に直結する経営者なら妥当な投資。

### Q: 途中でやめられる？
A: プラン変更・解約はいつでも可能。月単位課金。

### Q: 代行者が自分のAPIキーを使うのは問題ない？
A: X API利用規約上、**各エンドユーザーが自分のAPI credentialsを持つべき**。代行業務をする場合、クライアントごとにアカウント・API契約を分けるのが安全。
